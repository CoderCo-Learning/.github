# CoderCo Learning

**Learn DevOps by doing.** Hands-on labs, real-world projects, and production-grade infrastructure — not slides.

## What we teach

- **Containers & Kubernetes** — Docker, K8s, Helm, EKS, AKS
- **Infrastructure as Code** — Terraform, modules, state management
- **CI/CD** — Pipelines, GitOps, deployment strategies
- **Cloud Platforms** — AWS, Azure, GCP
- **System Design** — Architecture, scaling, reliability
- **Security** — Vault, secrets management, infrastructure hardening
